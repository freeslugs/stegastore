# StegaStore
Use Flickr's free 1-terabyte storage for storing files of any kind! 

# Install
```
$ git clone git@github.com:kzeng10/stegastore.git
$ npm install
```
Work in progress. Main functions in `main.js`, to be built into a full-fledged app (web?)
